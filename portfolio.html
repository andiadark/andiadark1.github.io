Operating Systems




 
  For this project I was tasked with developing a processor manager for a single-processor, multi-programming system. The process manager needed to include the tasks of the job scheduler and the process scheduler. The job scheduler will use a shortest time remaiming policy.
 I had to create a process control block class that included the following fields:

  • pid
  • state
  • arrival time
  • completition time
  • total burst
  • burst so far
  • priority

Project Requirements

  1. Create a PCB class.
  2. Create PCB getter and setter functions.
  3. Load the jobs queue.
  4. Use the algorithm for Shortest Time Remaining to processes the jobs in the jobs queue.
  5. Print out the contect of the completed processes queue.

Project Development: Capstone Semester 2


Project Development: Capstone Semester 2


  This course is in progress. The team was given a list of functional requirements that are due by the end of the semester. During several sprints we are given new requirements from the list to complete and showcase. As part of the functional requirements we had to create and Admin and Client sides within our app. Our Admins were hardcoded into our database and upon login needed to have the ability to perform tasks such as:

  • Activate, Deactivate and Ban users
  • Generate several reports
  • View and post to the social feed

When a client downloads the app, they are required to register as a new user, and then will be prompted to login. A client must be able to perform tasks such as:

  • View and post to the social feed
  • Show their search history
  • View and search for movies
  • Show messages from Admins
  • Take a survey to generate a list of movie or tv show recommendations
